# 路线图

## 0.1 项目骨架与空画布

- 初始化 Vite + React + TypeScript 项目。
- 建立 `core`、`canvas`、`manipulatives`、`features`、`components`、`utils` 目录。
- 创建中文首页和空 SVG 画布。
- 定义可 JSON 序列化的操作板文档模型。
- 配置构建、类型检查和基础测试。

## 0.2 基础画布交互

- 支持选择、拖拽、删除对象。
- 支持基础缩放和平移。
- 显示简单网格和画布坐标边界。
- 建立画布命中测试和坐标转换测试。

## 0.3 第一批数学教具

- 数字方块。
- 分数条。
- 基础几何图形。
- 简单计数器或筹码。
- 每类教具具备 JSON 默认值和 SVG 渲染器。

## 0.4 保存与读取

- 本地导出 JSON。
- 从 JSON 导入操作板。
- 加入 `schemaVersion` 兼容检查。
- 提供示例文件。

## 0.5 教师任务模板

- 定义任务模板 JSON 格式。
- 提供 3 到 5 个内置中文任务模板。
- 支持从模板创建操作板。
- 支持任务目标、提示语和初始教具对象。

## 0.6 课堂演示体验

- 增加演示模式。
- 增加对象锁定和隐藏辅助线。
- 优化大屏投影下的字号和对比度。

## 0.7 学生练习体验

- 加入简单的目标检查器。
- 提供即时反馈文本。
- 记录本地完成状态，不引入账号系统。

## 0.8 更多教具与主题

- 数轴。
- 坐标网格。
- 简单天平。
- 角度尺或量角器。
- 根据年龄段提供不同模板分组。

## 0.9 可访问性与稳定性

- 键盘操作基础支持。
- 提升色彩对比度。
- 增加关键流程测试。
- 完善错误提示和导入失败处理。

## 1.0 可发布 MVP

- 完成核心教具、画布、保存读取、任务模板闭环。
- 文档覆盖开发、数据格式和模板编写。
- 提供稳定的本地部署构建。
- 明确未来是否需要后端、学习记录和协作能力。
