# 教学活动

## 凑十

目标：理解一个数还差几个到 10。

操作：

1. 添加一个空十格阵。
2. 让学生点击格子填出 6 个点。
3. 观察属性面板里的“还差 4 个到 10”。
4. 继续补满十格阵，说出算式：6 + 4 = 10。

## 8 + 5 的进位

目标：用“先凑十，再加剩下的数”理解进位。

操作：

1. 添加一个 5 点十格阵，再点击补到 8 个点。
2. 添加一个数字方块 5，表示要加的 5。
3. 从 5 中拆出 2 个先补满十格阵，得到 10。
4. 剩下 3 个单独表示，合起来读作：8 + 5 = 10 + 3 = 13。

## 13 拆成 10 和 3

目标：理解 13 可以分解成一个十和 3 个一。

操作：

1. 添加一个 10 点十格阵。
2. 添加一个数字方块 3。
3. 同时选中十格阵和数字方块，观察右侧总数为 13。
4. 让学生说出分解：13 = 10 + 3。

# Fraction Bars Activities

## 1/2 = 2/4 的等值分数

目标：用长度相同的分数条理解等值分数。

操作：
1. 添加一个 1/2 分数条。
2. 添加一个自定义 2/4 分数条，放在 1/2 下方并左侧对齐。
3. 比较两个分数条的填充长度，观察它们覆盖的总长度相同。
4. 引导学生说出：1/2 和 2/4 表示同样大小的部分。

## 3/4 和 2/3 比较

目标：用可视化长度比较异分母分数。

操作：
1. 添加自定义 3/4 分数条。
2. 添加自定义 2/3 分数条，放在同一起点。
3. 观察两个分数条的填充长度。
4. 讨论为什么 3/4 比 2/3 大。

## 同分母分数加法

目标：理解同分母分数相加时分母不变、分子相加。

操作：
1. 添加一个 1/5 分数条。
2. 添加一个自定义 2/5 分数条。
3. 同时选中两个分数条，观察右侧属性面板显示每个小数值和同分母总和。
4. 读出算式：1/5 + 2/5 = 3/5。

# Geometry Tiles Activities

## 用小正方形拼面积

目标：理解面积可以用相同单位方块计数。

操作：
1. 添加多个正方形几何拼板。
2. 将正方形排成 3 行 4 列的长方形。
3. 数一数一共用了 12 个小正方形。
4. 选中拼出的长方形或单个正方形，比较属性面板中的面积估算。

## 用三角形拼正六边形

目标：观察正六边形可以分解为多个等边三角形。

操作：
1. 添加 6 个等边三角形几何拼板。
2. 旋转三角形，让它们的尖角围绕同一个中心。
3. 拼成一个正六边形轮廓。
4. 再添加一个正六边形放在旁边，对比两种表示方式。

## 找对称轴

目标：辨认常见几何图形的对称性。

操作：
1. 添加正方形、长方形、圆形、梯形和平行四边形。
2. 旋转或复制图形，尝试把图形沿想象中的直线对折。
3. 讨论哪些图形有对称轴，哪些没有。
4. 打开“显示顶点”，用顶点位置辅助判断。

## 比较周长和面积

目标：理解面积相近的图形，周长可能不同。

操作：
1. 添加一个正方形和一个长方形。
2. 在属性面板中调整宽度和高度，让两个图形面积接近。
3. 比较属性面板里的估算周长。
4. 讨论为什么面积相同或接近时，周长仍可能不同。
