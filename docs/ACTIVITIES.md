# 教学活动

## 凑十

目标：理解一个数还差几个到 10。

操作：

1. 添加一个空十格阵。
2. 让学生点击格子填出 6 个点。
3. 观察属性面板里的“还差 4 个到 10”。
4. 继续补满十格阵，说出算式：6 + 4 = 10。

## 8 + 5 的进位

目标：用“先凑十，再加剩下的数”理解进位。

操作：

1. 添加一个 5 点十格阵，再点击补到 8 个点。
2. 添加一个数字方块 5，表示要加的 5。
3. 从 5 中拆出 2 个先补满十格阵，得到 10。
4. 剩下 3 个单独表示，合起来读作：8 + 5 = 10 + 3 = 13。

## 13 拆成 10 和 3

目标：理解 13 可以分解成一个十和 3 个一。

操作：

1. 添加一个 10 点十格阵。
2. 添加一个数字方块 3。
3. 同时选中十格阵和数字方块，观察右侧总数为 13。
4. 让学生说出分解：13 = 10 + 3。

# Fraction Bars Activities

## 1/2 = 2/4 的等值分数

目标：用长度相同的分数条理解等值分数。

操作：
1. 添加一个 1/2 分数条。
2. 添加一个自定义 2/4 分数条，放在 1/2 下方并左侧对齐。
3. 比较两个分数条的填充长度，观察它们覆盖的总长度相同。
4. 引导学生说出：1/2 和 2/4 表示同样大小的部分。

## 3/4 和 2/3 比较

目标：用可视化长度比较异分母分数。

操作：
1. 添加自定义 3/4 分数条。
2. 添加自定义 2/3 分数条，放在同一起点。
3. 观察两个分数条的填充长度。
4. 讨论为什么 3/4 比 2/3 大。

## 同分母分数加法

目标：理解同分母分数相加时分母不变、分子相加。

操作：
1. 添加一个 1/5 分数条。
2. 添加一个自定义 2/5 分数条。
3. 同时选中两个分数条，观察右侧属性面板显示每个小数值和同分母总和。
4. 读出算式：1/5 + 2/5 = 3/5。

# Geometry Tiles Activities

## 用小正方形拼面积

目标：理解面积可以用相同单位方块计数。

操作：
1. 添加多个正方形几何拼板。
2. 将正方形排成 3 行 4 列的长方形。
3. 数一数一共用了 12 个小正方形。
4. 选中拼出的长方形或单个正方形，比较属性面板中的面积估算。

## 用三角形拼正六边形

目标：观察正六边形可以分解为多个等边三角形。

操作：
1. 添加 6 个等边三角形几何拼板。
2. 旋转三角形，让它们的尖角围绕同一个中心。
3. 拼成一个正六边形轮廓。
4. 再添加一个正六边形放在旁边，对比两种表示方式。

## 找对称轴

目标：辨认常见几何图形的对称性。

操作：
1. 添加正方形、长方形、圆形、梯形和平行四边形。
2. 旋转或复制图形，尝试把图形沿想象中的直线对折。
3. 讨论哪些图形有对称轴，哪些没有。
4. 打开“显示顶点”，用顶点位置辅助判断。

## 比较周长和面积

目标：理解面积相近的图形，周长可能不同。

操作：
1. 添加一个正方形和一个长方形。
2. 在属性面板中调整宽度和高度，让两个图形面积接近。
3. 比较属性面板里的估算周长。
4. 讨论为什么面积相同或接近时，周长仍可能不同。

# Balance Scale Activities

## 3 + 4 = 7

目标：用天平理解等式两边相等。
操作：
1. 添加一个空天平。
2. 添加数字方块 3 和 4，并同时选中它们和天平。
3. 点击“把选中数字方块总和设为左边”，让左边变成 7。
4. 将右边数值设为 7，观察天平回到平衡状态。

## x + 3 = 8 的直觉活动

目标：用“保持平衡”的直觉理解未知数。
操作：
1. 添加一个天平，把右边设为 8。
2. 把左边先设为 3，观察右边下降。
3. 提问：左边还需要多少才能平衡？
4. 把左边设为 8，说明缺少的部分可以记作 x，得到 x + 3 = 8 中 x = 5。

## 化学反应前后质量守恒类比活动

目标：把天平平衡类比为反应前后总质量不变。
操作：
1. 添加平衡天平 5 = 5。
2. 讨论左边代表反应前物质总质量，右边代表反应后物质总质量。
3. 调整任意一边数值，观察天平倾斜。
4. 回到相等数值，说明守恒时总量应保持一致。

# Algebra Tiles Activities

## 合并同类项

目标：用同形代数砖理解同类项可以合并。
操作：
1. 添加多个 +x、-x、+1、-1 代数砖。
2. 多选这些代数砖，观察属性面板中的合并表达式。
3. 把相同形状的砖排成一组，讨论为什么只有同类项能合并。
4. 移除一对 +1 和 -1，观察表达式如何变化。

## x + x + 1 + 1

目标：把重复的代数砖写成简化表达式。
操作：
1. 添加两个 +x 代数砖。
2. 添加两个 +1 代数砖。
3. 同时选中四块砖，读出属性面板中的表达式 2x + 2。
4. 讨论 2x + 2 和 x + x + 1 + 1 表示同一个总量。

## x² + 2x + 1 与正方形面积模型

目标：用面积模型直观看到完全平方结构。
操作：
1. 添加一个 +x² 代数砖。
2. 添加两个 +x 代数砖，分别放在大正方形的右侧和下侧。
3. 添加一个 +1 代数砖，放在右下角补成正方形。
4. 多选四块砖，读出 x² + 2x + 1，并讨论它可以看作边长 x + 1 的正方形面积。
